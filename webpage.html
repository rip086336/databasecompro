C Compiler



คำอธิบายโปรเจกต์

เนื่องจากในปัจจุบันข้อมูลนั้นมีมากมายหลายประเภท จึงใช้ทรัพยากรในการจัดเก็บมากตามปริมาณของข้อมูล จึงมีหลายคนพยายามคิดค้นนวัตกรรมใหม่ๆเพื่อลดการใช้ทรัพยากรที่ใช้จัดเก็บข้อมูลได้อย่างมีประสิทธิภาพ เป็นระเบียบ และง่ายต่อการใช้งาน

เราจึงได้นำโปรแกรมจัดเก็บข้อมูลมาต่อยอดให้ใช้สำหรับจัดเก็บข้อมูลของนักศึกษา โดยมีประเภทข้อมูล ดังนี้ชื่อ อายุ รหัสนักศึกษา คะแนนและเกรด โปรแกรมนี้สามารถจัดการข้อมูลได้โดยการ เพิ่ม ลบ แก้ไข และ แสดงผลข้อมูลเป็นตาราง สร้างความสะดวกสบายให้กับผู้ใช้งานเพราะไม่ต้องจัดเก็บด้วยตัวเอง และไม่เปลือง ทรัพยากรธรรมชาติ เพราะสามารถใช้คอมพิวเตอร์แทนกระดาษในการจัดเก็บได้เพิ่มความปลอดภัยให้กับข้อมูล โดยโปรแกรมของเรานั้นมีจุดเด่นที่ UI สบายตา ใช้ง่าย เป็นมิตรกับผู้ใช้งาน

โดยสาเหตุที่เราเลือกทำโครงงานนี้ เพราะว่าเราอยากศึกษาภาษาที่ใช้ในการพัฒนาโปรแกรม นั่นคือภาษา C โดยเราได้นำโปรแกรมจัดเก็บข้อมูลของคนอื่นมาต่อยอดและปรับแต่งใหม่ให้เป็นมิตรกับผู้ใช้งานยิ่งขึ้น โดยการ เพิ่มสีสัน เพิ่มตารางในการแสดงผลข้อมูล เปลี่ยนอัลกอริทึม ฯลฯ

ผลลัพธ์จากการพัฒนาโปรแกรมดังกล่าว ผู้พัฒนาทั้งหมด ได้ความรู้จากการพัฒนาโปรแกรมดังกล่าวเป็น อย่างมาก ทั้งด้าน หน้าบ้านและหลังบ้าน และอัลกอริทึมในการเขียนโปรแกรมเพิ่มเติม



คู่มือการใช้งาน

เริ่มจากการ compile code แล้วจะมีไฟล์ตระกูล .exe แล้วกดเข้าใช้แอพพลิเคชั่นได้เลย แอพพิลเคชั่นนี่ใช้ในการเก็บข้อมูลของนักศึกษาซึ่งมีฟังก์ชันให้เลือก 4 ฟังก์ชัน

1. ถ้าหากต้องการใส่ข้อมูลนักศึกษาในโปรแกรมให้กดเลข 1 บนคีบอร์ดต่อจากนั้น จะมีให้ใส่ ชื่อ อายุ เลขนักศึกษา และคะแนนที่ได้ และถ้าหากต้องการใส่ต่อให้กดปุ่ม 'Y' แต่ถ้าต้องการกลับไปหน้าเมนูให้กด 'M'

2. ถ้าหากต้องการเปิดดูข้อมูลที่ใส่เข้าไปในโปรแกรมให้กดเลข 2 บนคีบอร์ดและ จากนั้นจะมีตารางที่ใส่ข้อมูลที่เรากรอกขึ้นไปมาให้ดู และถ้าต้องการกลับหน้า เมนูให้กดปุ่ม 'M'

3. เป็นฟังก์ชันแก้ไขหรือ Modify ข้อมูลถ้าหากใช้ให้กดปุ่มเลข 3 บนคีบอร์ด และใส่รหัสนักศึกษาที่เราต้องการแก้ไขถ้าหากใส่ไม่ถูกต้องจะมีให้กด 'Y' เพื่อใส่อีกครั้ง และกด 'M' เพื่อกลับเมนูแต่ถ้าหากใส่ถูกต้องจะขึ้นให้ใส่ ชื่อ อายุ เลขนักศึกษา และ คะแนน และเมื่อแก้ไขเสร็จและอยากใส่เพิ่มให้กด 'Y' เพื่อแก้คนอื่นแต่ถ้ากลับเมนูให้กด 'M'

4. เป็นฟังก์ชันลบหรือ delete ข้อมูลถ้าหากต้องการใช้ให้กดปุ่มเลข 4 บนคีบอร์ด และใส่รหัสนักศึกษาที่เราต้องการลบทิ้งถ้าใส่ไม่ถูกจะมีให้กด 'Y' เพื่อใส่อีกครั้ง และกด 'M' เพื่อกลับเมนูแต่ถ้าหากใส่ถูกก็จะถามว่าถ้าอยากลบเพิ่มให้กดปุ่ม 'Y' แต่ถ้าไม่ต้องการให้กดปุ่ม 'M'

5. ถ้าหากเราต้องการออกจากแอพพลิเคชั่นก็สามารถทำได้โดยกดเลข 5 บนคีบอร์ด

Employee Record System

Employee Record System คือ Application ขยาดย่อมที่มี UI เป็น Terminal และสามารถสั่งการรับค่าจาก keyboard เท่านั้น โดยตัว Application สามารถการเก็บข้อมูลหรือสถิติต่างๆ ไม่ว่าจะเป็นเป็นข้อมูลประเภท ข้อความ จำนวนเต็มหรือจำนวนจริง และสามารถนำมาคำนวนเพิ่มเติมได้ ตัว Application ถูกเขียนมาด้วยภาษา C ทั้งหมด โดยตัว Application สามารถเพิ่มข้อมูล แก้ไข ลบ หรือแสดงข้อมูลในรูปของตารางได้ หากต้องการสร้างฐานข้อมูลที่มี category ก็สามารถนำไปต่อยอดได้เพิ่มขึ้นโดยใช้เพียงแค่ภาษา C ภาษาเดียวก็สามารถสร้างได้
